# mtl文件

三维模型处理会要读取.mtl文件来获得材质信息

.mtl文件（Material Library File）是材质库文件，描述的是物体的材质信息，ASCII存储，任何文本编辑器可以将其打开和编辑。一个.mtl文件可以包含一个或多个材质定义，对于每个材质都有其颜色，纹理和反射贴图的描述，应用于物体的表面和顶点。

```shell
 # 定义一个名为 'xxx'的材质
 newmtl xxx
 # 材质的环境光（ambient color）
 Ka 0 0 0
 # 散射光（diffuse color）用Kd
 Kd 0.784314 0.784314 0.784314
 # 镜面光（specular color）用Ks
 Ks 0 0 0
 # 折射值 可在0.001到10之间进行取值。若取值为1.0，光在通过物体的时候不发生弯曲。玻璃的折射率为1.5。
 Ni 1
 # 反射指数 定义了反射高光度。该值越高则高光越密集，一般取值范围在0~1000。
 Ns 400
 # 滤光透射率
 Tf 1 1 1
 # 渐隐指数描述 参数factor表示物体融入背景的数量，取值范围为0.0~1.0，取值为1.0表示完全不透明，取值为0.0时表示完全透明。
 d 1
 # 为漫反射指定颜色纹理文件
 map_Kd test_vt.bmp
```

```shell
newmtl B01
Ns 96.078431
Ka 1.000000 1.000000 1.000000
Kd 0.640000 0.640000 0.640000
Ks 0.000000 0.000000 0.000000
Ke 0.400000 0.400000 0.400000
Ni 1.000000
d 1.000000
illum 1
map_Kd http://www.xxx.com/demo/models/texture_X1/CO1_middle_mylar_backlight.png
```



| 关键词   | 含义（中文）                          | 说明                                                    |
| -------- | ------------------------------------- | ------------------------------------------------------- |
| newmtl   | 新建材质名称                          | 定义一个新的材质，后续参数都属于该材质                  |
| Ka       | 环境光颜色（Ambient Color）           | 材质在环境光下的颜色，通常为 RGB 三个浮点数             |
| Ke       | 自发光颜色（Emissive Color）          | 材质自身发光的颜色，不受光源影响，通常为 RGB 三个浮点数 |
| Kd       | 漫反射颜色（Diffuse Color）           | 材质在漫反射光照下的颜色，通常为 RGB 三个浮点数         |
| Ks       | 高光颜色（Specular Color）            | 材质高光部分的颜色，通常为 RGB 三个浮点数               |
| Ns       | 高光反射指数（Shininess）             | 材质高光的锐利程度，数值越大高光越集中                  |
| d        | 透明度（Alpha）                       | 材质透明度，1.0为不透明，0.0为完全透明                  |
| Tr       | 透明度（Alpha，反向）                 | 透明度，等价于 d，但数值与 d 相反（Tr=1-d）             |
| map_Ka   | 环境光贴图（Ambient Texture）         | 用于环境光的纹理贴图文件路径                            |
| map_Ke   | 自发光贴图（Emissive Texture）        | 用于自发光的纹理贴图文件路径                            |
| map_Kd   | 漫反射贴图（Diffuse Texture）         | 用于漫反射的纹理贴图文件路径                            |
| map_Ks   | 高光贴图（Specular Texture）          | 用于高光部分的纹理贴图文件路径                          |
| map_Ns   | 高光反射指数贴图（Shininess Texture） | 用于高光反射指数的纹理贴图文件路径                      |
| map_Bump | 法线贴图（Normal/Bump Texture）       | 用于法线或凹凸效果的纹理贴图文件路径                    |
| map_d    | 透明度贴图（Alpha Texture）           | 用于透明度的纹理贴图文件路径                            |
